# Research Desk Rules

## Citations

* Include source URLs whenever possible
* Prefer primary sources over secondary summaries
* For papers use:
  [title](https://arxiv.org/abs/{arxiv_id})

---

## Papers

* Use `paper_search` for research and literature questions
* Use `read_paper` after `paper_search`
* Never guess arXiv IDs
* If a paper is unavailable, explain why and use web sources

---

## Research Notes

* Save important findings to `notes/`

* Prefer updating existing notes over creating duplicates

* Use:

  * `append` for new findings
  * `replace` for revisions
  * `delete` for stale content

* Use lowercase filenames:

  * notes/flashattention.md
  * notes/q-learning.md

---

## Web Search

* Use `web_search` before `web_fetch`
* Avoid fetching more than 3 pages unless needed
* Summarize information before presenting it

---

## Tool Usage

* Read files before editing them
* Do not overwrite notes unnecessarily
* Use the most relevant tool available

---

## Tone

* Be concise
* Be factual
* Put long-form research into notes files
* Use bullet points when possible
